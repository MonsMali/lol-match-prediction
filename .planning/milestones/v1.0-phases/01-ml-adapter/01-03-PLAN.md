import resource
import os

# Measure baseline before any imports
baseline_kb = resource.getrusage(resource.RUSAGE_SELF).ru_maxrss

# Import and initialize adapter
from src.adapter import LoLDraftAdapter
adapter = LoLDraftAdapter()

# Measure after loading
loaded_kb = resource.getrusage(resource.RUSAGE_SELF).ru_maxrss
print(f"Baseline RSS: {baseline_kb / 1024:.1f} MB")
print(f"After load RSS: {loaded_kb / 1024:.1f} MB")
print(f"Delta: {(loaded_kb - baseline_kb) / 1024:.1f} MB")

# Run a prediction
from src.adapter.schemas import DraftInput
# Use real teams and champions from the training data (check adapter.valid_champions and adapter.valid_teams)
teams = list(adapter.valid_teams)[:2]
champs = list(adapter.valid_champions)[:20]

result = adapter.predict_from_draft(DraftInput(
    blue_team=teams[0], red_team=teams[1],
    blue_picks={"top": champs[0], "jungle": champs[1], "mid": champs[2], "bot": champs[3], "support": champs[4]},
    red_picks={"top": champs[5], "jungle": champs[6], "mid": champs[7], "bot": champs[8], "support": champs[9]},
    blue_bans=[champs[10], champs[11], champs[12], champs[13], champs[14]],
    red_bans=[champs[15], champs[16], champs[17], champs[18], champs[19]],
))

after_predict_kb = resource.getrusage(resource.RUSAGE_SELF).ru_maxrss
print(f"After predict RSS: {after_predict_kb / 1024:.1f} MB")
print(f"Prediction: blue={result.blue_win_prob:.4f}, red={result.red_win_prob:.4f}")
print(f"Sum: {result.blue_win_prob + result.red_win_prob:.6f}")

Then create **tests/test_adapter.py** with the following test functions (use pytest):

**test_adapter_singleton:**
- LoLDraftAdapter() is LoLDraftAdapter() (same instance)

**test_get_status:**
- Returns AdapterStatus with champion_count > 100, team_count > 10, memory_usage_mb > 0
- model_name and model_version are non-empty strings

**test_predict_from_draft_basic:**
- Create a DraftInput with valid teams and champions (use real names from adapter.valid_champions/valid_teams)
- Call predict_from_draft
- Assert blue_win_prob is between 0 and 1
- Assert red_win_prob is between 0 and 1
- Assert blue_win_prob + red_win_prob is approximately 1.0 (within 1e-6)
- Assert model_name and model_version are non-empty

**test_predict_swapped_teams_changes_result:**
- Run prediction with teams A vs B
- Run prediction with teams B vs A (swapped)
- Assert blue_win_prob differs between the two calls (probabilities are NOT symmetric)

**test_predict_not_degenerate:**
- Run predictions with 3 different valid drafts
- Assert not all blue_win_prob values are identical (would indicate broken features)
- Assert no prediction is exactly 0.5 (would indicate all features defaulting)

**test_validation_unknown_champion:**
- Create DraftInput with one invalid champion name
- Assert raises ValueError
- Assert error message contains "Unknown champion" and "Did you mean"

**test_validation_unknown_team:**
- Create DraftInput with invalid team name
- Assert raises ValueError
- Assert error message contains the invalid name

**test_validation_duplicate_champion:**
- Create DraftInput where one champion appears in both blue_picks and red_picks
- Assert raises ValueError mentioning "duplicate"

**test_validation_missing_role:**
- Create DraftInput with blue_picks missing the "support" role key
- Assert raises ValueError

**test_validation_wrong_ban_count:**
- Create DraftInput with only 3 blue bans
- Assert raises ValueError

**test_no_csv_loading:**
- Import src.adapter.adapter module
- Grep the source code string for "read_csv" -- assert it does NOT appear
- This is a static check that no CSV loading sneaks in

**test_memory_under_512mb:**
- Get adapter.get_status().memory_usage_mb
