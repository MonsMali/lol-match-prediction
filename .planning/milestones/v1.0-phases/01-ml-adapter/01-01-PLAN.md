  - `memory_usage_mb: float`
  - `champion_count: int`
  - `team_count: int`

**src/adapter/normalization.py:**
- `CHAMPION_ALIASES: dict[str, str]` -- mapping from common alternate names to canonical names found in the lookup dicts (from Task 1 audit findings). Must include at minimum: Wukong/MonkeyKing and Nunu variants. Add any others discovered in the audit.

  - Check alias map first (case-insensitive)

  - If no match, raise `ValueError` with fuzzy suggestions using `difflib.get_close_matches(name, valid_champions, n=3, cutoff=0.6)`

- `normalize_team_name(name: str, valid_teams: set[str]) -> str` function:
  - Case-insensitive match against valid teams
  - If no match, raise `ValueError` listing all valid teams
