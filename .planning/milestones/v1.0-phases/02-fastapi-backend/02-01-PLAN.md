**api/schemas.py** -- Pydantic v2 BaseModel classes:
- `TeamDraft(BaseModel)`: fields `top`, `jungle`, `mid`, `bot`, `support` (all `str`).
- `PredictRequest(BaseModel)`: fields `blue_team: str`, `red_team: str`, `blue_picks: TeamDraft`, `red_picks: TeamDraft`, `blue_bans: list[str]` (Field min_length=5, max_length=5), `red_bans: list[str]` (Field min_length=5, max_length=5), `patch: str | None = None`.
- `PredictResponse(BaseModel)`: fields `blue_win_probability: float`, `red_win_probability: float`. NO metadata, NO confidence, NO echoed draft (per CONTEXT.md decision).
- `ChampionInfo(BaseModel)`: fields `name: str`, `key: str` (internal key used in adapter), `image_url: str` (full DDragon CDN URL).
- `ChampionListResponse(BaseModel)`: field `champions: list[ChampionInfo]`.
- `TeamListResponse(BaseModel)`: field `teams: dict[str, list[str]]` (league -> team names).
- `HealthResponse(BaseModel)`: fields `status: str`, `model_name: str | None = None`, `model_version: str | None = None`, `champion_count: int | None = None`, `team_count: int | None = None`, `memory_usage_mb: float | None = None`.

**api/dependencies.py** -- Two dependency functions:
- `get_adapter(request: Request) -> LoLDraftAdapter`: Checks `request.app.state.model_ready`. If False, raises `HTTPException(status_code=503, detail="Model is loading")`. Otherwise returns `request.app.state.adapter`.
- `require_admin_token(authorization: str = Header(...))`: Reads `ADMIN_TOKEN` from `os.environ`. If not set or token does not match `f"Bearer {expected}"`, raises `HTTPException(status_code=401, detail="Invalid admin token")`.

**api/champion_mapping.py**:

- Constant `DDRAGON_BASE = f"https://ddragon.leagueoflegends.com/cdn/{DDRAGON_VERSION}/img/champion"`.
- Dict `DDRAGON_ID_MAP` mapping display names to DDragon asset IDs. Include at minimum: `"Wukong": "MonkeyKing"`, `"Nunu & Willump": "Nunu"`, `"Kai'Sa": "KaiSa"`, `"Kha'Zix": "Khazix"`, `"Cho'Gath": "Chogath"`, `"Vel'Koz": "Velkoz"`, `"Rek'Sai": "RekSai"`, `"Kog'Maw": "KogMaw"`, `"Bel'Veth": "Belveth"`, `"K'Sante": "KSante"`, `"LeBlanc": "Leblanc"`, `"Renata Glasc": "Renata"`.

**api/routers/health.py**:
- Create an `APIRouter` with no prefix (health lives at root `/health`).
- `GET /health` endpoint (async def). Reads `request.app.state`. If `model_ready` is True, call `adapter.get_status()` and return `HealthResponse` with all fields populated (status="ready", model_name, model_version, champion_count, team_count, memory_usage_mb rounded to 1 decimal). If `model_ready` is False, return `HealthResponse(status="loading")` with all optional fields as None.
- The health endpoint must ALWAYS respond, even during loading. It does NOT use the `get_adapter` dependency (which would raise 503).

**api/main.py**:
- Import `FastAPI`, `CORSMiddleware`, the `asynccontextmanager` from contextlib.
- Import `LoLDraftAdapter` from `src.adapter`.
- Create lifespan async context manager:
  1. Set `app.state.model_ready = False` and `app.state.adapter = None`.
  2. Try to instantiate `LoLDraftAdapter()`. On success, set `app.state.adapter` and `app.state.model_ready = True`. On exception, store the error string in `app.state.model_load_error`.
  3. `yield`.
  4. Shutdown: no cleanup needed.
- Create `app = FastAPI(title="LoL Draft Predictor", version="1.0.0", lifespan=lifespan)`.
- Add CORS middleware: `allow_origins=["http://localhost:5173", "http://localhost:3000"]`, `allow_credentials=False`, `allow_methods=["GET", "POST"]`, `allow_headers=["*"]`.
- Include the health router. (Other routers will be added in Plan 02 and 03.)
- Add a comment placeholder for predict, champions, teams, and admin routers (to be added in subsequent plans).

Install FastAPI: Run `pip install "fastapi[standard]" python-multipart` in the project's venv (or document it). Do NOT modify requirements.txt yet (Plan 03 handles production deps).
