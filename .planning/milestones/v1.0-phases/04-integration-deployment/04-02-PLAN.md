  <action>
1. Create render.yaml in the project root with this configuration:
   ```yaml
   services:
     - type: web
       name: lol-draft-predictor
       runtime: python
       plan: free
       buildCommand: pip install -r requirements-prod.txt && cd frontend && npm ci && npm run build
       startCommand: uvicorn api.main:app --host 0.0.0.0 --port $PORT
       envVars:
         - key: PYTHON_VERSION
           value: "3.12"
         - key: MODEL_UPLOAD_TOKEN
           generateValue: true
       autoDeploy: false
   ```

Key configuration details:
- `runtime: python` -- Render Python native runtime includes Node.js/npm in build environment
- `plan: free` -- Render free tier (512 MB RAM, 15-min spin-down)
- `buildCommand` -- Installs Python deps from prod file, then builds React SPA
- `startCommand` -- Single uvicorn process using $PORT env var (Render default is 10000)
- `PYTHON_VERSION: "3.12"` -- Match development environment
- `MODEL_UPLOAD_TOKEN: generateValue` -- Auto-generates a secret for the admin upload endpoint
- `autoDeploy: false` -- Prevents accidental deploys on every push

2. Do NOT create a Procfile -- render.yaml is preferred and more declarative.

3. Rebuild the frontend after creating WarmUpScreen (Task 1 must complete first):
   - cd frontend && npm run build
   - This ensures frontend/dist/ includes the warm-up screen
