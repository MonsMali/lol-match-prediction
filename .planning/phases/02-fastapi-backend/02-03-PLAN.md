  <action>
**api/routers/admin.py**:
- Create `APIRouter(prefix="/api/admin", tags=["admin"])`.
- `POST /upload-model` endpoint (async def). Dependencies: `Depends(require_admin_token)`.
- Accept three `UploadFile` parameters: `model_file`, `scaler_file`, `encoders_file`.
- Implementation steps:
  1. Create a temporary directory (use `tempfile.mkdtemp()`).
  2. Save uploaded files to temp dir as `best_model.joblib`, `scaler.joblib`, `encoders.joblib`.
  3. **Validate the new artifacts:** Attempt to load them with `joblib.load()`. If any fail to load, return 422 with detail explaining which file is invalid.
  4. **Run a test prediction:** Create a new `LoLDraftAdapter` instance pointing to the temp directory (NOT the singleton -- create a fresh instance by temporarily bypassing the singleton pattern). Build a dummy draft dict using champions from the new adapter's `valid_champions` and teams from `valid_teams`. Call `predict_from_draft()` on the test adapter. If the test prediction fails, return 422 with the error details.
  5. **Swap artifacts:** Copy the validated files from temp dir to the production directory (`models/production/`). Also copy the lookup dicts (meta_strength, synergies, team_perf) if they were included, though the primary upload is model+scaler+encoders.
  6. **Reset the singleton:** Set `LoLDraftAdapter._instance = None` and `LoLDraftAdapter._initialized` removal. Then create a new `LoLDraftAdapter()` which will load from the updated production files. Update `request.app.state.adapter` with the new instance.
  7. Clean up the temp directory.
  8. Return `{"status": "success", "message": "Model swapped successfully"}`.
- **Zero downtime:** The old adapter remains in `app.state.adapter` during the entire validation process. Only after successful validation does the swap happen. The swap itself (pointer assignment) is atomic in CPython due to the GIL.
- **Error handling:** Wrap the entire process in try/except. On any failure, clean up temp files and return appropriate error. Never leave `app.state.adapter` in a broken state.
