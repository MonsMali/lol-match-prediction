Create src/types/index.ts with TypeScript types matching the Phase 2 API schemas exactly:
- `Side = 'blue' | 'red'`
- `DraftAction = 'ban' | 'pick'`
- `DraftMode = 'live' | 'bulk'`
- `SeriesFormat = 'single' | 'bo3' | 'bo5'`
- `Role = 'top' | 'jungle' | 'mid' | 'bot' | 'support'`
- `ChampionInfo = { name: string; key: string; image_url: string }`
- `TeamDraft = { top: string; jungle: string; mid: string; bot: string; support: string }`
- `PredictRequest = { blue_team: string; red_team: string; blue_picks: TeamDraft; red_picks: TeamDraft; blue_bans: string[]; red_bans: string[]; patch?: string | null }`
- `PredictResponse = { blue_win_probability: number; red_win_probability: number }`
- `DraftStep = { team: Side; action: DraftAction; slotIndex: number }`

Create src/api/client.ts:
- Export `API_BASE` using `import.meta.env.VITE_API_URL || ''` (empty string because Vite proxy forwards /api to backend in dev; in production SPA is served from same origin)
- Export `apiFetch(path, options?)` wrapper that prepends API_BASE and throws on non-OK responses

Create src/api/champions.ts:
- Export `useChampions()` using TanStack Query v5 single-object API
- queryKey: `['champions']`, staleTime: `Infinity` (static data)
- Fetches `GET /api/champions`, returns `ChampionInfo[]`

Create src/api/teams.ts:
- Export `useTeams()` with queryKey: `['teams']`, staleTime: `Infinity`
- Fetches `GET /api/teams`, returns `Record<string, string[]>`

Create src/api/predict.ts:
- Export `usePrediction()` as `useMutation`
- mutationFn takes `PredictRequest`, POSTs to `/api/predict`, returns `PredictResponse`

Create src/store/draftStore.ts with Zustand:
- Export the 20-step DRAFT_SEQUENCE constant (exact sequence from RESEARCH.md)
- State shape: `blueBans`, `redBans`, `bluePicks`, `redPicks` (all `(string | null)[]` length 5), `blueTeam`, `redTeam` (string | null), `blueRoles`, `redRoles` (Record<Role, string | null>), `mode` (DraftMode, default 'live'), `currentStep` (number, default 0), `activeSlot` (for bulk mode: `{ team, action, index } | null`), `seriesFormat` (default 'single'), `seriesScore` ({ blue: 0, red: 0 }), `currentGame` (1)
- Computed getter: `usedChampions()` returns `Set<string>` of all non-null picks and bans
- Actions: `selectChampion(name)` -- in live mode, places into slot determined by `currentStep` and auto-advances; in bulk mode, places into `activeSlot`
- `setActiveSlot(team, action, index)` -- for bulk mode
- `setTeam(side, name)`, `setRole(side, role, champion)`
- `setMode(mode)` -- when switching from bulk to live, recalculate `currentStep` by scanning DRAFT_SEQUENCE for first unfilled slot
- `setSeriesFormat(format)`, `recordGameResult(winner)` -- increments score, advances game number
- `resetDraft()` -- clears picks/bans/roles but preserves teams and series state
- `resetAll()` -- full reset including teams and series
- `isComplete()` -- derived: all 20 slots filled
- `isDraftReady()` -- derived: all slots filled AND both teams selected AND all roles assigned
